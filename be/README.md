# Backend

Backend project for the application.

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Running the Server

```bash
npm start
```

For development mode with hot reload:

```bash
npm run dev
```

## Available Scripts

- `npm start` - Start the server
- `npm run dev` - Start in development mode
- `npm test` - Run tests
